# Real-mart-Sales-Dashboard
Designed an interactive Power BI dashboard using the Real Mart Sales dataset. Visualized key metrics like total sales, quantity sold, gross margin %, and payment modes. Analyzed sales trends by city, gender, month, and customer type to identify performance patterns.
